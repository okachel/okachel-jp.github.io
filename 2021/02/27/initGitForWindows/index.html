WindowsでGithubを使用する

前書き

2021年2月27日時点の記事なので、見ている時期によってはリンク先などの情報が変わっているかもしれません。

その時は申し訳ございませんが、読み替えてご閲覧ください。

現状、Hexoでの画像の挿入方法を試していますが上手くいかないので画像が挿入できるようになるまでは
文章のみの説明となります。
読みづらいとは思いますがよろしくお願いします。

対象

Gitを使用していない人。

方法

  1. Githubのアカウントを作成する。

    ※既にGithubのアカウントを持っているならこの作業は不要です。

    • Google ChromeやInternet ExploreなどのブラウザでGithubのサイトを開きます。リンク→Github

    • 右上にある「Sign up」を押して必要事項を入力し、登録を完了させてください。

  2. Git for Windowsをインストールします。

    WindowsではデフォルトでGitを使用できないので、インストール作業が必要です。

    • Git for Windowsのサイトに行き、Downloadのボタンを押してインストーラーをダウンロードします。

      ダウンロード先→Git for Windows

    • インストーラーを起動します。Windowsならエクスプローラーのダウンロードフォルダに入っているはずです。

    • インストーラーは起動後すべてnextで良いです。私の環境では問題は出ていません。

  3. Gitの設定ファイルを編集します。

    • 「Windowsキー + R → cmdを入力」でコマンドプロンプトを開きます。

    • 以下のコマンドを打ち込みGithubに登録したユーザー名とメールアドレスを設定します。

      git config --global user.name "ユーザー名"

      git confit --global user.email "メールアドレス"

以上で設定は終了です。
